    FontLoader

    Extends:

    core/AbstractLoader~AbstractLoader → FontLoader

    FontLoader is solely porting webfontloader as a loader

    The main goal of this loader is to trigger a ready state only when the font(s) are ready to be parsed to avoid FOUT artifacts

    See:

    • https://github.com/typekit/webfontloader
